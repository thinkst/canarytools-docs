#### Example

:::: tabs :options="{ useUrlFragment: false }"

::: tab "cURL"

``` bash

  -d auth_token=EXAMPLE_AUTH_TOKEN \
  -d node_id=EXAMPLE_NODE_ID
```

:::

``` python
import requests
